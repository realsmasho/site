Privacy Policy

Last Revised: 25 August 2024

  1. Introduction

    We are dedicated to ensuring everyone feels comfortable knowing what we know when you use the Meme Wars Application! If you care about this, then please continue reading. This Privacy Policy outlines the type of information we collect, how we collect it, and who has access to it.

    Unless otherwise stated, all definitions in the Terms of Service apply to this Privacy Policy.

  2. Information Collection
    1. Global

      Nothing is recorded outside of the Server unless it has not been authorized (for more information, refer to Section 2.3: Unauthorized Use). Data that Server managers set up (custom prompts and custom prefixes) can be accessed outside of the Server by us, but unless we have proof or reasonable suspicion that they violate the Server’s rules and/or Discord's Terms of Service, we will never access this data.

    2. Server Events

      None of the data in this section is ever recorded outside of the Server.

      Server owners, or anyone with the Manage Server permission in a server with the Application, can use the Application’s text commands to allow it to record certain events inside the Server, which are disabled by default:

      • /echo messages. If you execute the /echo command and /echo logs are enabled, Meme Wars will record the message, your username and ID, and the channel the message was sent in or to. Messages are automatically blocked from sending and logged if you attempt to send a message to an Important channel, or if the Application detects "@everyone", "@here", or an invite link to a Discord server.
      • Welcome messages. If you join a server Meme Wars is a member of and welcome logs are enabled, It will log your profile, whether it is your first time joining the Server, whether your account is at least seven days old, and, if you joined the Meme Wars server, whether the App's welcome DM could be sent to you. This is simply to help the managers of the Server know who joined.
      • Leave messages. If you leave a server Meme Wars is a member of and leave logs are enabled, It will log your profile and ID, the number of messages you sent, and, if you were banned from the Server via the App, the reason for the ban, if any is provided. This is simply to help the managers of the Server know who left.

      Anyone with the Manage Server permission can opt-out by running the respective text command in any Channel the Application has access to:

      • To opt-out of /echo logs: (prefix) reset log
      • To opt-out of welcome messages: (prefix) reset joinchannel
      • To opt-out of leave messages: (prefix) reset leavechannel

      For the commands above, replace "(prefix)" with your Server's custom prefix—if none is set, use "!mw", as it is the default prefix.

    3. Unauthorized Use

      At the time of this document’s latest revision, Meme Wars is marked as private. Adding the App to any Server without our prior consent is strictly forbidden. If It is somehow added to a Server without prior consent and a slash command is used, Meme Wars will record the command used, the user who executed it, and essential information about the Server wherein the command was executed. This allows us to better understand the Server the Application was added to by providing essential information without requiring us to join it. For actions we may then take, refer to Section 3: Restrictions in the Terms of Service.

    4. Methods of Collection

      By using the Application’s Services, you agree that your username and user ID may be recorded automatically if you execute a command in an unauthorized Server or are the owner of the Server in that same scenario. If you do not consent, please do not use the Application’s Services.

    5. Use of Data

      We do not share any information with anyone outside of data requests (for more information, refer to Section 5: Contact Information).

  3. Data Security

    All data can only be accessed by @smasho on Discord.

  4. Changes to the Privacy Policy

    We may make revisions to this Privacy Policy at any time.

  5. Contact Information

    If you have any questions or concerns regarding this Privacy Policy, our data-handling practices, or if you would like to request the information we have collected about you, please contact @smasho on Discord or via email at help@smasho.xyz.
