Privacy Policy

Last Revised: 8 August 2024

  1. Introduction

    We are dedicated to ensuring everyone feels comfortable knowing what we know when you use the Economy Wars Application! This Privacy Policy serves to outline the type of information the App collects, how it is collected, and who has access to it.

    All definitions stated in the Terms of Service apply here as well.

  2. Information Collection
    1. Global

      The Economy Wars Application stores only basic information like what Pets you own, what level they are, what their name is, and so forth.

      Some actions, such as modifying a Pet or setting a new Goal, may be logged to private channels for reference in the event of a bug in the code.

    2. Unauthorized Use

      Economy Wars is a private Application. Adding the App to any Server without our prior consent is strictly forbidden. If It is somehow added to a Server and a slash command is used, Economy Wars will send us the command that was ran, the user who ran it, and some info about the Server wherein the command was executed. This allows us to beter understand the Server the Application was added to by providing essential information without requiring us to join it. See the Terms of Service, section 3. Restrictions for actions we may then take.

    3. Methods of Collection

      By using the Application’s Services, you agree that your username and user ID may be recorded automatically if you execute a command in an unauthorized Server or are the owner of the Server in that same scenario. If you do not consent, please do not use the Application’s Services.

    4. Use of Data

      We do not share any information with anyone outside of data requests (see 4. Your Rights to Access).

  3. Data Security

    All data can only be accessed by @smasho on Discord.

  4. Your Rights to Access

    You can request information by contacting @smasho.

  5. Changes to the Privacy Policy

    We may make revisions to this Privacy Policy at any time.

  6. Contact Information

    If you have questions or concerns regarding the Privacy Policy or any data-handling practices outlined in this document, please contact @smasho.
